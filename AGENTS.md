# AGENTS.md - PLI Labs

This repo is for Proprietary Legal Intelligence / PLI Labs work.

## Scope

- Treat PLI as a separate business and technical workstream from family court
  automation, private law-practice operations, and other legal repos.
- Use this repo for PLI Labs research, open-source artifact planning, model
  release materials, eval notes, tooling prototypes, and related lab work.
- Keep public-release materials cleanly separated from private source materials
  and business knowledge stored outside this repository.

## GitHub Identity

- Do not assume this repo uses the same GitHub account, organization, SSH key, or
  remote as any non-PLI repo.
- Use the PLI GitHub identity for this repo. Do not switch it to the user's
  other GitHub remotes or SSH identities without explicit instruction from the
  repo owner.

## Data Boundaries

- Do not add private client files, privileged work product, attorney-client
  communications, or closed-client materials to this public repo.
- PLI Labs public-release work should be cleanly separable from private
  law-practice materials.
